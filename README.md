# VW-T5 USB-C Dashboard fastcharger

## Overview
12v USB-c fastcharger that replaces a dashboard switch blank on VW T5

## Features
- simple fitment
- 35w USB-C fast charging

## Hardware
- USB-C QC3 fastcharge board
- PETG filament

## Software
- Fusion 360
